# Warehouse Daily Dispatch Report for Odoo 16

Developed by **Faizal Emam** ([itdelhi.in](https://itdelhi.in))

## Overview
This module provides a detailed inventory dispatch report designed for high-volume retail operations like **Blue Sky Coffee**. It tracks SRO (Stock Return/Order) deliveries and provides real-time costing analysis for Finance.

## Features
- **Dynamic Costing**: Fetches accurate product costs from `ir_property` and `product_template`.
- **Branch Reporting**: Filter and group by warehouse branches.
- **Finance Ready**: Optimized Pivot and Tree views for daily auditing.
- **UoM Integration**: Displays Units of Measure alongside product names for clarity.

## Technical Stack
- Odoo 16.0
- PostgreSQL (SQL Views)
- Python / XML
